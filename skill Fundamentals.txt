STILL function do:

----------
STILL Function: compute a data value 
- assign the return value to a skill variable
	mySelectedData = geGetSelSet()

- pass the return value to another SKILL function
	TrPrintInfo ( geGetSelSet() )

	virtuoso &
	virtuoso -nograph
	virtuoso -replay <log-file>

Variables
	naming		Line_Count1
	assignment	x = 5
	retrieval	x
	

	

notes：	retrieval：
	It typically means:
	Reading design objects (cells, nets, shapes)
	Querying properties/attributes
	Navigating hierarchy (top cell → instances → pins → nets)

Notes: first letter of variable can not be digit or question mark
	variable name case sensitive
