Function Arguments
	Argument
	- expected data type of the argument
	- required/ Optional/  a keyword 
types
	required arguments with no optional arguments	
	keyword arguments with no required and no optional arguments	

See list of arguments for a given function
	arglist( 'printf ) => (t_string \@optional g_general "tg") 
		; pass a string followed by any type of arg

SKILL function argument examples
Required arguments: 
must provide each required argument
In prescribed order
	hiSetFilterOptions( g_accelInput g_prompt g_output greturnAccel
			g_typedReturn g_errorOutput g_warnOutput)
Optional Arguments:
Each optional argument has default value
If provide optional argument, must provide all the preceding optional arguments
	view( t_file [g_boxSpec][g_title][g_autoUpdate ][l_iconPosition])

Keyword Arguments: 
when provide a keyword argument you must preface it with the name of the formal argument
You can provide keyword arguments in any order
	geOpen(	?window		w_windowId
		?lib		t_lib
		?cell		t_cell
		?view		t_view
		?viewType	t_viewType
		?mode		t_mode	) =>t / nil
